== Description ==

myCred-H5P lets you award points to your users on H5P interactive activities. Encourage and increase user engagement by adding gamification element to your H5P multimedia content. Also, recognized individual achievements by awarding badges on completion of H5P content.

= myCred H5P Features =
Here are the most notable features of myCred H5P plugin.

* Award points on completion of any interactive content.
* Award points on completion of specific content.
* Hooks available.
* Award achievements on completion of any multimedia content.
* Award achievements on completion of specific multimedia content.
* Points reward on achieving the required percentage of any interactive content.
* Points reward on achieving the required percentage of specific interactive content.
* Logs generation for reward points on H5P interactive content.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred H5P
3. Find myCred H5P and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
